```tsx showLineNumbers
  return <FlipClock size="lg" variant="secondary" />;
}
```

## API Reference

### FlipClock

| Prop         | Type                                                                | Default     | Description                                                            |
| ------------ | ------------------------------------------------------------------- | ----------- | ---------------------------------------------------------------------- |
| `size`       | `"sm" \| "md" \| "lg" \| "xl"`                                      | `"md"`      | Controls the overall size of the clock and digits.                     |
| `variant`    | `"default" \| "secondary" \| "destructive" \| "outline" \| "muted"` | `"default"` | Visual style variant.                                                  |
| `countdown`  | `boolean`                                                           | `false`     | Enables countdown mode instead of real-time clock.                     |
| `targetDate` | `Date`                                                              | `undefined` | Target date used when `countdown` is enabled. Supports up to 3 digits. |
| `showDays`   | `"always" \| "never" \| "auto"`                                     | `"auto"`    | Controls the visibility of the days unit in countdown mode.            |
| `className`  | `string`                                                            | `undefined` | Additional Tailwind classes.                                           |

### FlipUnit (internal)

This component is used internally by `FlipClock` to render and animate individual digits. It is not intended for direct use.

## Notes

- Animations are handled via CSS keyframes bundled with the component.
- The component is client-side only.
- Accessible updates are announced using `aria-live`.
- When `showDays` is set to `"auto"`, the days unit is displayed only when the remaining time is greater than zero days.
