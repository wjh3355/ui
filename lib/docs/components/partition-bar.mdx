## Usage

```tsx showLineNumbers
import {
  PartitionBar,
  PartitionBarSegment,
  PartitionBarSegmentTitle,
  PartitionBarSegmentValue
} from "@/components/ui/partition-bar";
```

```tsx showLineNumbers
<PartitionBar size="md">
  <PartitionBarSegment num={3}>
    <PartitionBarSegmentTitle>Apples</PartitionBarSegmentTitle>
    <PartitionBarSegmentValue>30%</PartitionBarSegmentValue>
  </PartitionBarSegment>

  <PartitionBarSegment num={7} variant="secondary">
    <PartitionBarSegmentTitle>Oranges</PartitionBarSegmentTitle>
    <PartitionBarSegmentValue>70%</PartitionBarSegmentValue>
  </PartitionBarSegment>
</PartitionBar>
```

## API Reference

### PartitionBar

| Prop       | Type              | Default     | Description                                                                              |
| ---------- | ----------------- | ----------- | ---------------------------------------------------------------------------------------- |
| `gap`      | `number`          | `1`         | The gap between partition bar segments, in terms of tailwindcss spacing units.           |
| `children` | `React.ReactNode` | `undefined` | The child elements of the partition bar, which must be `PartitionBarSegment` components. |

### PartitionBarSegment

| Prop        | Type                                                                | Default     | Description                                                                                                                                                                      |
| ----------- | ------------------------------------------------------------------- | ----------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `num`       | `number`                                                            | `0`         | The numerical value representing the size of this segment relative to others. The width of this segment is calculated based on this value relative to the total of all segments. |
| `variant`   | `'default' \| 'secondary' \| 'destructive' \| 'outline' \| 'muted'` | `'default'` | The visual style variant of the partition bar segment.                                                                                                                           |
| `alignment` | `'left' \| 'center' \| 'right'`                                     | `'center'`  | The text alignment within the partition bar segment.                                                                                                                             |
| `children`  | `React.ReactNode`                                                   | `undefined` | The child elements of the partition bar segment, which can must be `PartitionBarSegmentTitle` and `PartitionBarSegmentValue` components.                                         |

### Manually Setting Width

By default, the entire partition bar fills the width of its container. You can manually set the width of the partition bar using standard CSS width properties.

<ComponentPreview name="partition-bar-width-demo" align="start" />

### Custom Colours

You may also customize the colours of the partition bar segments using standard CSS properties. Note that using the `className` prop of `PartitionBarSegment` will apply it to the <strong>coloured bar</strong> only. To customize the text colours, you can use the `className` prop on `PartitionBarSegmentTitle` and `PartitionBarSegmentValue`.
